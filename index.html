§1. Что такое функция и зачем она нужна

Представь, что ты каждый день утром варишь чай. Действия одни и те же: налить воду → вскипятить → положить заварку → подождать.

Функция - это «рецепт», который ты записываешь один раз, а потом используешь сколько угодно раз, просто называя его по имени.

Аналогия из жизни

Без функцииС функцией
Каждый раз пишешь:
«Налей воду, вскипяти, положи заварку…»		 Один раз записываешь рецепт «Свари чай».
Потом просто говоришь: свари_чай()

Словарь терминов

ТерминЧто означаетПример
def Ключевое слово «определить функцию» (от англ. define) def greet():
Параметр Имя «входа» при определении функции def greet(name): — name параметр
Аргумент Конкретное значение, которое передаём при вызове greet("Маша") — "Маша" аргумент
return Команда «верни результат наружу» return a + b
Вызов Использование функции по имени greet("Маша")
💡 Запомни: функцию нужно сначала определить (написать рецепт), а потом вызвать (использовать рецепт).

Посмотри на самый простой пример - функция, которая печатает приветствие:

Схема

│ │ │ │
│ │ │ └── двоеточие — начинает блок тела
│ │ └──── скобки — место для параметров

⚠️ Частая ошибка: забыть двоеточие : в конце строки def. Python выдаст SyntaxError - синтаксическую ошибку.
⚠️ Отступы обязательны! Тело функции (строки внутри неё) должно начинаться с отступа - 4 пробела. Без отступа Python не поймёт, что строка принадлежит функции.

§4. Возвращаем результат: return

print() просто показывает текст на экране. А return - возвращает значение, и его можно сохранить в переменную или использовать дальше.

❌ Только print (не возвращает)

Вариант A
1 def add(a, b):
2 print(a + b)
3
4 x = add(3, 5)
5 print(x) # None 😢

✅ С return (возвращает)

Вариант Б
1 def add(a, b):
2 return a + b
3
4 x = add(3, 5)
5 print(x) # 8
💡 Главное отличие:
print() → показывает на экране, но результат «пропадает».
return → передаёт результат наружу, и ты можешь его сохранить в переменную.

Построчный разбор функции с return

📄 return_пример.py
1 def multiply(a, b):
2 result = a * b
3 return result
4
5 answer = multiply(4, 7)
6 print("Ответ:", answer)

§5. Значения по умолчанию

Иногда ты хочешь, чтобы параметр имел «стандартное» значение, которое используется, если аргумент не передали.

значения_по_умолчанию.py
1 def power(x, n=2):
2 return x ** n
3
4 print(power(5)) # 25 (5 ** 2)
5 print(power(5, 3)) # 125 (5 ** 3)
6 print(power(2, 10)) # 1024 (2 ** 10)
